[제약조건]
NOT NULL : NULL 값 허용 X
UNIQUE : 중복 값 X
PRIMARK KEY(PK) : 고유식별키(행을). NOT NULL + UNIQUE
FOREIGN KEY(FK) : PK를 다른 테이블에서 사용. (값 참조) 참조 무결성 보장
CHECK : Y or N 으로 값 지정.
DEFAULT : 기본값 지정

: Optional - null 값 허용
: Mandatory - FK로 참조
: Identify - 복합 PK..?

pk 값은 bigint로 설정.
