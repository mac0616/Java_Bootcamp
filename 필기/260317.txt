* 완벽한 프롬프트의 4대 구성 요소
1. 역할 부여하기
ex) 너는 10년 경력의 백엔드 개발자야. 자바와 스프링 기반의 웹 어플리케이션 개발 전문성이 있어.
=> 답변 깊이, 전문성 향상 / 해당 분야 전문용어 및 관점 반영 / 맞춤형 통찰력 제공

2. 지시 명확히 하기
ex)
before: LMS 관련 기능 만들어줘
after : LMS 프로젝트에서 강의 등록 기능 만들고자해. 관리자가 강의를 등록해야
하며, 입력 정보는 강의명, 강의 설명, 강사명이야 이를 참고해서 작성해줘.
=> 좋은 지시는 다음 요소를 포함한다.
1) 행동 동사 : 작성하라, 요약하라, 분석하라, 비교하라 등의 명확한 행동 지시
2) 대상 : 무엇에 대해 작업할 지 명확히 지정
3) 구체성 : 모호함을 제거하고 구체적인 작업 범위 제시

3. 맥락 제공하기
before: 마케팅 이메일 작성해줘.
after: 우리는 20~30대 직장인 대상 온라인~~장점은~점이야.

4. 제약 설정하기
- 형식 제약, 톤앤매너 제약, 길이 제약, 키워드 제약
